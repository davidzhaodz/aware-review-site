The Discovery

An Instagram reel by Edward Sturm (Compact Keywords) showed that queries with 7 or more words in Google Search Console correlate with AI-assisted searches - people using ChatGPT, Perplexity, or Google AI Overviews to craft their queries. We tested this hypothesis on AWARE's GSC data.

AWARE is visible to AI searchers - but captures zero clicks

Of 1,556 query+page rows in the last 90 days, 117 rows (7.5%) are AI-assisted queries with 7 or more words. These generate 359 monthly impressions across 65 topic clusters. The click count across every single one: zero. AWARE shows up in results when people ask AI-crafted questions about scalp care, professional conditioners, and salon treatments - but there is no content optimized to convert those impressions into visits.

How the Engine Works

The engine runs daily at 06:45 as a Node.js script (zero Claude tokens). It pulls GSC data, identifies AI-assisted queries, groups them into topic clusters, scores each cluster's content opportunity, and persists everything to Supabase.
1

Pull GSC Data

Fetches 90 days of search analytics from Google Search Console via REST API. Uses query + page dimensions to see which AWARE pages rank for each query. Today: 1,556 rows, 1,225 unique queries, 67 pages.

2

Filter AI-Assisted Queries

Counts words per query. Anything with 7 or more words is flagged as likely AI-assisted. This threshold catches conversational queries from ChatGPT users, AI Overview reformulations, and LLM-coached searches. Research by Search Engine Land confirms 8+ word queries have grown 7x since AI Overviews launched.

3

Cluster by Topic

Groups queries that share 3+ significant words (after removing 91 stopwords). Example: "american academy of dermatology dandruff shampoo leave on 5 minutes" and "american academy of dermatology dandruff treatment ketoconazole" share 4 words and form one cluster. This produces 65 clusters from 117 queries - 50 single-query clusters and 15 multi-query clusters.

4

Score Opportunities

Each cluster gets an opportunity score: impressions x (1 / avg_position) x (1 - CTR). This prioritizes clusters with high visibility (impressions), close to page 1 (low position number), and room for improvement (low CTR). Since all AI queries have 0% CTR, the score simplifies to impressions / position.

5

Content Gap Analysis

Compares cluster topics against H1 tags from AWARE's 12 published blog articles. Clusters matching existing content are tagged optimize_existing. Unmatched clusters with score > 5.0 are tagged write_new. The rest are monitor. Result: 5 write-new, 4 optimize, 56 monitor.

6

Persist + Brief

All clusters upserted to Supabase (seo_query_clusters table), engine run logged, and a markdown brief generated at ~/.config/aware-hair/briefs/YYYY-MM-DD/. The brief feeds into the morning briefing and overnight signal system.

Daily Automation Flow

The engine runs autonomously as part of AWARE's cron-based automation system. No manual intervention required - opportunities surface in the morning brief and content tasks are auto-generated overnight.
06:30
GSC Data Pull
gsc-performance.sh collects raw Search Console data (rankings, impressions, clicks) and writes to daily briefs directory
06:45
PUBLISHER: SEO AI Engine
seo-engine.mjs runs: pull GSC (90 days, 5000 rows) -> filter 7+ word queries -> cluster by shared words -> score opportunities -> check against 12 blog articles -> upsert 65 clusters to Supabase -> generate markdown brief
06:50
SENTINEL: B2B Health Engine
Parallel engine checks salon reorder health and inventory stockout projections
07:00
Morning Brief
Compiles all engine outputs + Shopify + Klaviyo into a single daily briefing. SEO AI opportunities appear alongside revenue and email metrics.
20:00
Signal Check + Task Generation
generate-tasks.sh reads the SEO brief and checks 5 signal thresholds. If a cluster scores > 5.0 with no content, it auto-creates a content brief task. If a ranking improves 3+ positions, it creates a momentum alert.
21:00
Overnight Dispatch
Auto-generated tasks executed: content briefs drafted, CTR optimization suggestions written, momentum alerts filed. All output saved as files for David's review - never published autonomously.

Signal Thresholds

Five automated triggers that convert engine data into overnight tasks. Each signal has a tier: Tier 0 = observe/report only, Tier 1 = draft to file for review.
SignalWhen It FiresWhat It DoesTier
seo_ai_opportunity Cluster score > 5.0, no existing AWARE content Auto-generates a content brief (keyword, outline, CTA) as a draft file Tier 1
seo_ranking_improvement Published article improves 3+ positions week-over-week Momentum alert with suggestions for quick wins to push further Tier 0
seo_zero_click_spike 50+ impressions, 0 clicks for 2+ consecutive weeks Drafts improved title tags and meta descriptions for the ranking page Tier 1
seo_strike_distance Position 8-15 with 200+ impressions, existing content Optimize existing article for page 1 breakthrough Tier 1
