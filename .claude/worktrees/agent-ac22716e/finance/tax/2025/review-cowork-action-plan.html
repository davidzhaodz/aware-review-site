A2X has already posted all Shopify revenue as journal entries. The ~230 bank feed items for Shopify payouts must be MATCHED to these entries, never posted as new revenue.

Procedure:
  1. Go to Transactions > Bank Transactions > Shopify Balance
  2. For each "Shopify Payments Payout": click Match (not Add)
  3. Verify date and amount match the A2X journal entry within $0.01
  4. If no match suggested: flag for investigation
  5. Never post as new revenue - A2X already recorded the income
Watch out: QBO suggests "Merchant Processing Fees" for 34 payouts ($11,736). These are NOT fees - they are sales payouts that need A2X matching. Do not accept the QBO suggestion.

How Claude Helps

Claude can query the 462 A2X journal entries by date and amount to help identify which JE matches each payout. This is especially useful when QBO doesn't auto-suggest a match or when payouts span multiple settlement days.
