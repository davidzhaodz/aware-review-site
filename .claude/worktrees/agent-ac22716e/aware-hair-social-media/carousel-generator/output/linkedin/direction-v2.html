Design System v2 - What Changed
Editorial techniques applied:
• Kicker system - Tracked uppercase label + gold hairline rule on every slide (magazine eyebrow)
• Three-tier type stack - 11px kicker / 64px+ headline / 18px deck (at 1080px). 5.8x size jump = drama
• Ghost typography - Oversized slide numbers (01, 02...) at 3.5% opacity as background texture
• Corner brackets - Le Labo laboratory motif on specimen/structured slides
• Tonal hierarchy - Same teal (#102B26) at 100% / 55% / 35% / 20% opacity. No new colors introduced.
• Vertical rule quotes - 1.5px gold left border on pull quotes and key statements
• Dot grid motif - Subtle reticle pattern in one quadrant (scientific positioning)
• Layout archetype rotation - 5 distinct layouts: Cover Statement, Pull Quote, Specimen Card, List Article, Data Point
• Diamond dividers - ◆ in bylines and kickers (from Instagram design guide)

What's consistent across every slide (the 6-element glue):
• Cream (#f4efe9) background with grain texture
• Folio counter bottom-right (DM Sans, tracked)
• Gold accent line bottom edge
• Kicker + hairline rule at content top
• Orpheus Pro headlines / DM Sans body / JetBrains Mono labels
• 15% edge margins (80px at 1080px)
