The System

Playwright rendering
pipeline

01

Render Scripts

11 custom .mjs files, each a self-contained slide deck. Full HTML/CSS per slide, rendered to PNG via headless Chromium at exact Instagram dimensions.

02

3 Output Formats

Feed (1080x1350), Story (1080x1920), Ad (1080x1080). Each render script outputs all three from the same source HTML.

03

Design Families

Dark-modern (deep teal overlays on photography), Light-editorial (cream backgrounds), Light-clean (product-focused). Consistent brand tokens across all.

04

Photo Library

37 curated stock photos sourced via Unsplash search API - editorial portraits, environmental textures, water macro, city atmosphere, botanical close-ups. Each carousel uses unique imagery.

05

Batch Factory

Weekly JSON config defines carousels, slides, and variants. batch-content-factory.mjs renders everything to date-stamped output directories automatically.

06

Review Dashboard

Single HTML file (carousel-dashboard.html) displays all 10 carousels with scroll strips and click-to-zoom lightbox. One view, all content.

Production Workflow

From idea to
publication-ready

01

Define

Weekly batch JSON specifies carousel concepts, slide content, photo assignments, and output formats.

weekly-batches/2026-03-24.json
02

Render

Playwright launches headless Chromium, renders each slide as full HTML/CSS, screenshots at exact platform dimensions.

node batch-content-factory.mjs
03

Review

Interactive HTML dashboard shows all carousels with scroll strips and lightbox zoom. Click any slide to inspect at full resolution.

carousel-dashboard.html
04

Export

PNGs go directly to Instagram. LinkedIn gets multi-page PDF via export script. Ad variants ready for Meta upload.

node export-linkedin-pdf.mjs
Quality Standard

Every detail
is intentional

Iterative Refinement

Each carousel went through 4-7 versions. Photo-topic matching, overlay calibration, font sizing, CTA variation - all refined through systematic audit passes.

Unique Imagery

37 curated photos with zero cross-carousel portrait reuse. Each carousel has its own closing image, cover, and topic-matched interior slides.

Editorial Typography

Georgia serif headlines for authority, DM Sans body for clarity. Font sizes optimized for mobile-first readability at Instagram feed scale.

Brand Consistency

Deep teal, cream, and gold tokens enforced across all 61 slides. No generic templates - every slide is custom HTML/CSS.

Photo-Topic Match

Showerheads for water content. Cityscapes for air quality. Pollen macro for allergens. Sunset silhouettes for UV. Every background tells the story.

Multi-Format Ready

Feed (3:4), Story (9:16), and Ad (1:1) variants from the same source. LinkedIn PDF export for professional carousel documents.
