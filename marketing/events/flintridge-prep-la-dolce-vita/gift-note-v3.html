Changes from Critique

Copy

75 words, 3 paragraphs
34 words, 2 lines + sign-off

"The art of living well" echoes La Dolce Vita without literal translation. Product described in one line.

Typography

8px body (6pt print)
11.5-13px body (8.6-9.7pt print)

Orpheus Pro only (DM Sans for utility). Minimum 10px across entire card.

Borders

Double border + 4 corner ornaments
Single 0.75px gold frame + edge painting

Edge painting (simulated as outer border) does the luxury work.

Promo Code

"Use code FLINTRIDGE at checkout"
Removed. Ad handles commerce.

QR reframed as "complimentary scalp analysis" - extends the gift rather than asking for money.

Sign-off

"With compliments, AWARE Hair"
"With love for Flintridge Prep, Tina & Jade" in Orpheus italic

Personal names humanize the note. Brand logo is at the top.

Botanical

10-14% opacity ghostly vectors
20% opacity, larger, promoted to hero

Still recommend replacing SVGs with commissioned botanical art for final print.

Logos

SVG placeholder monogram + text wordmark
Real AWARE wordmark (front) + real monogram (back)

Each logo appears once. No redundancy.

Back Strategy

QR + promo + tagline + event label
Monogram + gift extension + QR + URL only

"Your gift includes a complimentary scalp analysis" adds value instead of asking for a purchase.
